# PR-11: Strategy Builder + Pulse Dashboard

## Objective
Build the final two tabs: (1) Strategy Builder for model portfolio construction with FM overrides and allocation visualization, and (2) Pulse Dashboard as the command center that bridges MarketPulse signals into MF-specific daily actions. After this PR, all 6 frontend tabs are functional.

---

## PART A: Strategy Builder

### APIs Consumed

- `POST /api/v1/strategies` — create strategy
- `GET /api/v1/strategies` — list strategies
- `GET /api/v1/strategies/{id}` — strategy detail
- `PUT /api/v1/strategies/{id}` — update strategy
- `POST /api/v1/strategies/{id}/backtest` — backtest strategy
- `GET /api/v1/strategies/{id}/backtests` — historical backtests
- `POST /api/v1/overrides` — create FM override
- `GET /api/v1/overrides` — list active overrides
- `DELETE /api/v1/overrides/{id}` — deactivate override
- `GET /api/v1/funds?search=&limit=20` — fund search for adding to strategy
- `GET /api/v1/lens/scores/{mstar_id}` — lens scores for each fund in strategy
- `POST /api/v1/holdings/overlap` — overlap check for strategy funds

#### File 2: `web/src/components/strategy/StrategyEditor.jsx`

Full-screen strategy editor with three panels:

**Left panel (280px): Fund Selector**
- Search bar to find and add funds
- Each result shows: name, category, mini 6-score bar, 1Y return
- Click "+" to add fund to strategy
- Currently added funds listed below with remove (×) button

**Center panel: Allocation Treemap**
- D3 treemap visualization showing the model portfolio
- Each fund = a rectangle. Size = allocation weight. Color = average lens quality (green=high, amber=mid, red=low)
- Fund name + weight % displayed in each rectangle
- **Interactive:** Drag rectangle edges to resize allocation weight (or use number inputs)
- Below treemap: summary stats — total allocation (must = 100%), weighted avg lens scores, estimated expense ratio

**Right panel (280px): FM Overrides**
- Active overrides list with badge colors
- "+ New Override" button opens form:
  - Override type: FUND_BOOST, FUND_SUPPRESS, CATEGORY_TILT, SECTOR_VIEW
  - Target: fund search or category dropdown or sector dropdown
  - Direction: POSITIVE / NEGATIVE / NEUTRAL
  - Magnitude: 1–5 slider
  - Rationale: text area (required, min 10 chars)
  - Expiry: date picker (optional)
- Active overrides show as colored indicators on the treemap

#### File 3: `web/src/components/strategy/AllocationTreemap.jsx`

D3 treemap for portfolio allocation.

**Design:**
- Squarified tiling algorithm
- Each fund rectangle shows: fund name (bold, 13px), allocation % (large, 20px), category (small, 11px)
- Color: gradient from slate-200 (low lens quality) through teal-200 (mid) to teal-600 (high)
- Fund rectangles have 2px gap between them
- Category grouping option: toggle to group by broad_category (Equity/Debt/Hybrid), showing nested rectangles
- Hover: tooltip with full fund details + all 6 lens scores
- Click: select fund → highlights it → shows detail in the right panel

**Allocation editing:**
- Each fund has a number input below the treemap for precise weight entry
- Total must equal 100% — show warning if != 100%
- Lock icon: lock a fund's weight (won't change when others adjust)

#### File 4: `web/src/components/strategy/OverridePanel.jsx`

FM override management panel:
- List of active overrides with: type badge, target name, direction arrow (↑/↓/→), magnitude stars (1-5), rationale preview, expiry date
- Click override → expand to show full rationale + edit/deactivate buttons

#### File 8: `web/src/components/dashboard/SignalActionCards.jsx`

The core innovation of the dashboard. Translates MarketPulse signals into MF-specific actions.

**Logic (computed client-side from MarketPulse data):**
- If breadth > 55%: "Equity deployment favorable" → show top 3 Return LEADER funds
- If breadth < 40%: "Caution — reduce equity exposure" → show high-risk funds to review
- If a sector enters Leading quadrant: "Sector opportunity" → show top funds for that sector
- If sentiment < 30: "Market fear — contrarian opportunity for long-term SIPs"
- If VIX > 25: "Volatility high — pause lumpsum, continue SIPs"

Each card: colored left border (green/amber/red), signal description, recommended action, fund chips (clickable → navigate to fund360), action buttons.

**If MarketPulse is offline:** Show "Signals unavailable — MarketPulse connection required" with a retry button.
