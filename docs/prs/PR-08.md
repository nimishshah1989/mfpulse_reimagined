# PR-08: Fund 360° — Six-Lens Radar + Compare + Holdings

## Objective
Build the Fund 360° deep-dive page with six-lens radar chart, fund comparison mode, holdings breakdown, performance charting, and historical lens evolution. This is the page users land on when they click "Deep Dive" from Universe Explorer.

---

## APIs Consumed

- `GET /api/v1/funds/{mstar_id}` — full fund detail (master + returns + risk + ranks + portfolio + holdings + sectors)
- `GET /api/v1/lens/scores/{mstar_id}` — six lens scores + classes + headline
- `GET /api/v1/lens/scores/{mstar_id}/history` — lens score evolution over time
- `GET /api/v1/funds/{mstar_id}/nav?period=1y|3y|5y|max` — NAV time series
- `GET /api/v1/funds/{mstar_id}/holdings?top=10` — top holdings
- `GET /api/v1/funds/{mstar_id}/sectors` — sector exposure breakdown
- `GET /api/v1/funds/{mstar_id}/peers` — peer comparison
- `GET /api/v1/funds/{mstar_id}/risk` — risk stats history
- `POST /api/v1/holdings/overlap` — overlap analysis for compare mode

### File 2: `web/src/components/fund360/RadarChart.jsx`

D3.js six-axis spider/radar chart. 

**Design:**
- 6 axes equally spaced (60° apart), labeled: Return, Risk, Consistency, Alpha, Efficiency, Resilience
- Scale 0–100 on each axis, with gridlines at 20, 40, 60, 80
- Fund's scores plotted as a filled polygon with teal-500 fill at 25% opacity, teal-600 stroke 2px
- Each vertex has a circle dot (r=5) with the score number next to it
- Center of chart shows the fund's headline tag in small text

**Compare mode:**
- Up to 3 funds overlaid on the same radar
- Fund 1: teal, Fund 2: amber, Fund 3: purple (all at 20% opacity fills)
- Legend below showing fund names with color swatches
- Hover a vertex → tooltip showing all funds' scores on that lens

**Size:** 320×320px, responsive.

### File 3: `web/src/components/fund360/PerformanceChart.jsx`

Recharts line chart for NAV time series.

- NAV line in teal-600, area fill with gradient
- Period selector tabs: 1M, 3M, 6M, 1Y, 3Y, 5Y, Max (default: 1Y)
- Hover crosshair showing date + NAV value
- Y-axis formatted with ₹ prefix
- X-axis: date labels appropriate to period (daily for 1M, monthly for 1Y, yearly for 5Y+)

### File 4: `web/src/components/fund360/ReturnsTable.jsx`

Three-row comparison table:
- Row 1: Fund returns (1M, 3M, 6M, YTD, 1Y, 2Y, 3Y, 5Y, 7Y, 10Y, SI)
- Row 2: Category average returns (same periods)
- Row 3: Difference (Fund - Category), color-coded green if positive, red if negative

All numbers: font-mono, formatPct(), green/red coloring.

### File 5: `web/src/components/fund360/HoldingsTable.jsx`

Top 10 holdings table: Rank, Name, ISIN, Sector, Weight%. Progress bar for weight. Click a holding name → opens new tab with Google search for that stock (simple external link).

### File 6: `web/src/components/fund360/SectorDonut.jsx`

Recharts PieChart/donut showing 11 Morningstar sector allocations. Inner radius 60%. Legend on right. Colors: one per sector from the chart palette. Hover shows sector name + exact percentage.

### File 7: `web/src/components/fund360/RiskStatsGrid.jsx`

Grid layout showing risk metrics across timeframes:
- Rows: Sharpe, Sortino, Alpha, Beta, StdDev, MaxDrawdown, Treynor, InfoRatio
- Columns: 1Y, 3Y, 5Y
- Cell color intensity: green for good values, red for bad (Sharpe > 1 = green, < 0.5 = red, etc.)
- Missing values show "—"

### File 8: `web/src/components/fund360/LensHistory.jsx`

6 inline sparklines (one per lens) showing how the score has changed over available history. Each sparkline: 80px wide, 24px tall, teal-500 stroke. Label on left: "Return ↑" or "Risk ↓" with direction arrow showing trend.

### File 9: `web/src/components/fund360/PeerTable.jsx`

Table of top 10 peers in the same SEBI category, sorted by default on 1Y return. Columns: Fund Name, 1Y Return, 3Y Return, Sharpe 3Y, each lens score. The current fund is highlighted with a teal-50 background row. Click any peer → navigates to Fund 360° for that peer.

### File 10: `web/src/components/fund360/CompareMode.jsx`

Compare mode overlay/panel. User clicks "Compare" → a search bar appears to add fund 2 (and optionally fund 3). Once added:
- Radar chart shows overlaid polygons
- Returns table adds columns for each fund
- Risk stats show side by side
- Overlap analysis shown if 2+ equity funds (from POST /holdings/overlap)

### File 11: Navigation integration

- Universe Explorer "Deep Dive →" button navigates to `/fund360?fund={mstar_id}`
- Fund 360° "Simulate →" navigates to `/simulation?fund={mstar_id}`
- Fund 360° "Add to Strategy →" navigates to `/strategy?add={mstar_id}`
- Peer table click navigates to `/fund360?fund={peer_mstar_id}`
